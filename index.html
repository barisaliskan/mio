Teknik · RAG

Cihazda semantik hafıza.

core/ai/rag-pipeline.ts
async retrieve(query, { topK = 5, moduleType }) { // 1) Soru cihazda embed → 1024-boyut (BGE-M3) const qv = await embeddingEngine.embed(query); // 2) Kosinüs benzerliği ile aday anılar (eşik 0.6) let hits = await vectorStore.search(qv, topK * 2, 0.6); // 3) Güncellik (30g) + modül alakası (×1.15) hits = hits.map(applyRecencyAndModuleBoost); // 4) En iyi 5 anı → sistem promptuna bağlam return hits.sort(byScore).slice(0, topK); }

RAG — retrieve tasarımı

Soru ve anılar aynı vektör uzayında değerlendirilir; her şey cihazda, milisaniyeler içinde.

  • BGE-M3, 1024 boyut, kosinüs benzerliği
  • Güncellik (30g yarılanma) + modül boost (×1.15) ile yeniden sıralama
  • 0.6 eşiği gürültüyü eler; en iyi 5 anı sistem promptuna girer
  • 512 token yanıta ayrılır; prompt kalan 4K bütçeye sığdırılır
